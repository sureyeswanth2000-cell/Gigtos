# Gigto React App

This is the starting repository for migrating the Gigto project to React. It currently contains a minimal `create-react-app`-like structure.

## Getting Started

1. `cd react-app`
2. `npm install` (or `yarn`)
3. `npm start`

Replace the placeholder `App.js` with components mirroring the functionality of the existing plain HTML/JS pages:

- Header / Footer / Menu
- Authentication pages
- Service listing and booking flows
- User dashboard (bookings)
- Admin dashboard with real-time updates

The Firebase configuration can be copied from `../firebase.js` and integrated using the Firebase JS SDK.

This workspace will eventually become the production codebase. For now it's just a skeleton to begin development.
